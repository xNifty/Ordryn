Keyboard Shortcuts

Shortcuts apply on the home page task list. Click the page (not the address bar) first if keys seem ignored.

After deleting tasks, an undo toast appears for up to 120 seconds.

nNew task (open sidebar)
/Focus search
EscClose sidebar or modal
j / kMove between task rows
Enter / eEdit focused task
dDelete focused task
xToggle complete on focused task
? or Shift+/Show this help
