About {{.SiteName}}

{{.SiteName}} is a self-hosted task manager built with Go, PostgreSQL, and HTMX. It is designed for individuals and small teams who want a simple, fast todo app without SaaS lock-in.

Features

  • Per-user tasks with projects, tags, due dates, favorites, priority levels, and drag-and-drop ordering
  • Search, project filters, tag filters, status filters (all / incomplete / complete), and due-date quick filters (today, overdue, this week, no date)
  • Markdown task descriptions with truncated list view and expand-in-place
  • Starred tasks appear in a pinned section on page 1 and are excluded from pagination counts
  • Priority badges (Low / Medium / High) and optional sort-by-priority view
  • User profiles with timezone and pagination preferences
  • Invite-only registration and role-based permissions
  • Dark and light themes
  • Responsive layout for desktop and mobile

Self-hosting

Configure your database and session key via .env, run migrations on startup, and serve the app behind HTTPS in production. See the project README for setup steps.

Source and documentation are available on the project's GitHub repository (linked in the footer).
